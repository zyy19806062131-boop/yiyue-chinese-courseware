# Render Deployment

Official Render service:

```text
yiyue-Chinese-courseware
```

Live URL:

<https://yiyue-chinese-courseware.onrender.com/>

## Blueprint

`render.yaml` defines a Python web service:

```yaml
services:
  - type: web
    name: yiyue-Chinese-courseware
    runtime: python
    buildCommand: pip install -r requirements.txt
    startCommand: python server.py
    healthCheckPath: /healthz
```

Do not change this service back to `static_site`; Render does not support
changing an existing service type in place.

## Environment Variables

Required:

- `SESSION_SECRET`: generated by Render from `render.yaml`.

Optional:

- `ADMIN_PASSWORD`: plaintext admin password override.
- `ADMIN_PASSWORD_HASH`: SHA-256 admin password hash override.
- `TRAINER_DATA_JSON`: optional tone trainer data override.
- `ACCESS_CODES_JSON`: optional tone trainer access code data.

If no admin password environment variable is set, the app uses the default
admin password hash for `19920811`.

## Legacy Services

The active Render workspace should only keep `yiyue-Chinese-courseware` active.
Legacy `niki-hskk-*` duplicate services can stay suspended.
